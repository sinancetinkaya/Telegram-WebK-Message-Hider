# Telegram-WebK-Message-Hider
A Greasemonkey/Violentmonkey script that hides messages from unwanted users in Telegram(WebK) groups

Example Values JSON
```
{
    "1234567890": {
      "name": "John Doe",
      "reason": "Annoying"
    },
    "9876543210": {
      "name": "Karen",
      "reason": "Life sucker"
    }
}
```
